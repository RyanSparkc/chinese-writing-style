# 中文寫作風格指南插件

為 Claude Code 提供清晰、簡潔、專業的中文寫作規則與技巧。

## 功能

- 句子結構原則（主動語態、肯定句式、刪除贅字等）
- 標點符號規則（全形標點、引號使用等）
- 常見贅字與誤用詞彙指南
- 中英混排規則
- 技術文件寫作指南
- 數字與單位規範

## 安裝

此插件已安裝在本機，重啟 Claude Code 即可使用。

## 使用方式

當你需要撰寫或編輯中文文字時，技能會自動提供寫作建議。

### 適用場景

- 技術文件、README
- commit 訊息、PR 描述
- 錯誤訊息、UI 文案
- 報告、摘要
